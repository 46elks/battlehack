Apini

The issue with virtually every mobile payment platform is that none is omnipresent there is either plattform lock-in or dependance on nationality or other arbitrary factor.

Apini solves this by being built on widely used protocols, using text messages and HTTP with easy payments using any credit card through PayPal and BrainTree.

With Apini the payee quickly initializes a payment with a single test message and the payer only has to submit their payment details on the link they get through SMS

Apini is built using Python, flask and microWSGI powered by a Postgres database on the serverside. We are using the 46elks API to connect with the phone network and send the text messages.
